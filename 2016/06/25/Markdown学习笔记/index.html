Markdown学习笔记

2016-06-25
Blog

Markdown是一种轻量级的标记语言(用Latex写了一学期作业的我感动地哭了，从作用上来说就是能让我们把重点放在内容上，排版只要通过极少量的语法就能完成.

标题

标题总共分为6级，将标题的内容加在#和空格的后面即可，#表示一级、##表示二级，以此类推

链接

按照[文字](链接)就可以为[]里面的文字增加链接了. e.g. ChrisyayuのBlog

列表

用*，+，-开头后加空格就可以建立无序列表，有序列表的话，开头就用数字和点(233

  • One
  • Two
  • Three

文字引用

用>和空格开头 e.g.

Hello World!

代码块

如果是在一行里面的话，用`代码`就行了 e.g.

#define != ==

要一整个代码块，那就在行开头用四个缩进空格 e.g.

public class Cat
{
  private string name;
  private int age;
  ....
}

插入图像

用![描述](链接地址) e.g.

  • Markdown
